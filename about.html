Về tôi

Xin chào! Mình là Khoa, sinh viên ngành Công Nghệ Thông Tin. Mình có niềm đam mê đặc biệt với dữ liệu và hệ thống lớn.

Một số thuật ngữ mình thường làm việc:

Apache Superset
Nền tảng khám phá và trực quan hóa dữ liệu mã nguồn mở cấp doanh nghiệp.
Data Lakehouse
Kiến trúc dữ liệu hiện đại kết hợp tính linh hoạt của Data Lake và quản lý của Data Warehouse.
SQL
Ngôn ngữ truy vấn có cấu trúc dùng để giao tiếp với cơ sở dữ liệu.

Kỹ năng chuyên môn

  • Truy vấn và phân tích dữ liệu (SQL)
  • Xây dựng Dashboard (Apache Superset)
  • Lập trình Web (HTML, CSS)
  • Sử dụng Git và GitHub

Lộ trình học tập học kỳ này

  1. Nắm vững kiến thức Cơ sở dữ liệu phân tán.
  2. Hoàn thiện dự án Lập trình trực quan đúng hạn.
  3. Trau dồi và nâng cao kỹ năng giao tiếp tiếng Trung.
